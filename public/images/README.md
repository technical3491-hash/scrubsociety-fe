# Auth Page Images

Please add 3 images to this directory for the login and register page carousel:

- `auth-1.jpg` - First carousel image (recommended: healthcare professionals, medical team, or medical facilities)
- `auth-2.jpg` - Second carousel image
- `auth-3.jpg` - Third carousel image

**Recommended image specifications:**
- Format: JPG or PNG
- Aspect Ratio: 16:9 or 4:3 (landscape orientation works best)
- Size: Recommended 1920x1080px or larger for high-quality display
- Content: Healthcare-related images that align with ScrubSocietyAI branding

The carousel will automatically cycle through these images on the login and register pages.
